CENOP Performance Dashboard

This directory contains the nightly performance artifacts: perf_history.csv, perf_plot.png, and perf_history.html. The plot shows elapsed run time for short representative runs over time and a rolling median for trend smoothing.

Quick guide

  • Elapsed: wall-clock seconds for the short run (default: 500 ticks, 500 porpoise).
  • Baseline: the reference value used to detect regressions in CI.
  • CI threshold: PRs fail when elapsed > baseline * 1.25 + 2s (25% slower + 2s tolerance).

Baseline update

Nightly aggregator updates baseline only when the median of the last 7 successful runs improves by at least 3% (conservative policy). Updates appear as PRs for review.
